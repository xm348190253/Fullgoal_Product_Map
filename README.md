# 中欧基金 · 产品全景

中欧基金 530+ 只产品的可视化面板。机构销售内部视图——一屏看完投资类型分布、风险收益矩阵、近期业绩与产品明细。

数据全部来自外部公开接口（天天基金 pingzhongdata + 腾讯财经），无任何内部口径数据。

## 本地运行

直接双击 `index.html` 即可在 Edge / Chrome 中打开。图表组件 ECharts 已内置本地副本 `echarts.min.js`，**离线 / 内网也能正常出图**；业绩数据仍需联网从天天基金公开接口拉取（断网时业绩列显示 `--`，但页面框架与图表正常）。

ECharts 加载顺序：本地副本 `echarts.min.js` 优先 → 失败再依次降级到 4 个 CDN 镜像（bootcdn 国内 → jsdelivr → unpkg → 百度静态资源），任一连通即可。

## 数据来源

| 模块 | 接口 |
|---|---|
| 近期业绩 / 单位净值 / 规模 / 基金经理 | 天天基金 `pingzhongdata` 公开数据集 |
| 投资类型三级分类 | 天天基金（东方财富）官方 `type` 字段 |
| 市场指数行情 | 腾讯财经 `qt.gtimg.cn` |
| 中欧产品池 | 离线提取自天天基金 `fundcode_search.js`，~530 只 |

公开接口偶发限流时，未拉到业绩的产品对应字段显示 `--`，不再回退到任何 mock 数据。

## 功能总览

### 三级筛选
- **一级（大类）**：股票型 / 混合型 / 债券型 / 货币型 / QDII / FOF
- **二级（策略）**：如混合型 → 偏股 / 灵活配置 / 偏债
- **三级（细分）**：如纯债 → 长期纯债 / 中短债

切换一级会重置二级、三级；点 Treemap 的任意区块可下钻定位到三级。

### 检索 · 自选 · 导出（机构销售实战）
- **全站搜索**：顶部搜索框按 基金名 / 代码 / 基金经理 实时过滤（与三级筛选叠加生效）
- **自选清单**：表格首列 ☆ 星标，浏览器本地持久化；「☆ 自选」开关只看自选产品
- **清除筛选**：任一筛选 / 搜索激活时出现「✕ 清除筛选」，一键重置
- **导出 CSV**：导出当前筛选结果（含数据截至日期 + 免责声明），UTF-8 BOM 保证 Excel 中文不乱码
- **手动刷新**：顶部「↻」清本地缓存并重新拉取当日最新业绩 / 行情

### 主视图
- **风险收益矩阵**（气泡图）：X = 最大回撤，Y = 近一年收益，气泡大小 = 规模，颜色 = 一级大类
- **投资类型三级规模分布**（Treemap）：一级 → 二级 → 三级 三层嵌套
- **产品明细表**：产品名 / 经理 / 投资类型 / 规模 / 昨日 / 近一周 / 近一月 / 近一年 / 最大回撤 / 夏普 / 三级细分

### KPI 卡片
- 总管理规模：基于已披露规模的产品之和（子标签说明覆盖率）
- 产品数量：当前筛选下的总条数
- 平均近一年收益率：基于已加载业绩的产品平均
- 数据完整度：已加载业绩 / 当前列表

### 抽屉详情
点击任意产品 → 右侧抽屉滑出（两个 Tab）：
- **概览**：近期业绩横条 + 核心指标 + 基金经理 + 真实历史净值走势（来自 pingzhongdata）+ 近期动态（仅真实事件：净值更新 / 机构持仓披露）
- **同类对比**：市场同类竞品对比表 + 净值叠加图 + 4 维雷达图（近一年 / 近一月 / 规模 / 夏普）+ 中欧内部平替

> 注：旧版「业绩归因」Tab 是按固定权重把年化收益拆成 β/α/久期/信用 的**合成示意数据**（并非真实归因），已整体移除——本面板只展示来自公开接口的真实数据。

### 数据健康面板
顶部铃铛按钮 → 列出所有数据合理性自检异常（夏普/回撤/规模/年化超区间等）。

## 加载行为

启动后台批量拉取所有"主份额"产品（~250 只）的 pingzhongdata，12 并发逐批，单只 8 秒超时，连续失败会自动降速（熔断退避），总加载预算 120 秒上限。预计 30–60 秒拉完全部，期间表格可正常浏览、未完成的行显示 `--`。业绩缓存按「自然日」失效（跨日强制刷新，保证净值时效）。

切换"显示份额变体（A/C/E…）"开关会再补拉剩余的 ~280 只非主份额。
